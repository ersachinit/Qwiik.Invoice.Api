## 4. What AI Got Wrong

Some AI-generated suggestions were intentionally not used or were modified after review.

Examples include:

* Suggestions to introduce additional abstraction layers such as repositories, which were ultimately removed because Entity Framework Core already provides repository-like functionality through DbContext.
* Suggestions that could have increased complexity beyond the scope of the assignment.
* Suggestions involving additional frameworks or patterns that were not necessary for demonstrating the required functionality.

These suggestions were reviewed and simplified to keep the solution aligned with the assignment requirements and time constraints.
